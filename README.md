En este trabajo se desarrollaron varios ejercicios enfocados en la
aplicación de técnicas de ciencia de datos y machine learning utilizando
Python.

El objetivo principal fue:

-   Diseñar problemas tipo laboratorio
-   Implementar generadores de casos de uso con datos aleatorios
-   Desarrollar soluciones utilizando librerías como pandas, numpy y
    sklearn
-   Verificar automáticamente los resultados

Los ejercicios abordan diferentes áreas como: - Series de tiempo -
Procesamiento de texto (NLP) - Clustering - Clasificación
